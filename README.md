# SaaS_Product_Growth_Analysis
Purely Based on Numpy

This project analyzes performance of 6 SaaS products over 4 weeks using randomly generated data.

Features:
- Normalize website visits, ad spend, signups, cancellations
- Compute NetGrowthScore and WeightedScore
- Identify top 25% and bottom 25% products
- Row-wise total and column-wise average performance
- Maximum, minimum scores and above-average performing products
